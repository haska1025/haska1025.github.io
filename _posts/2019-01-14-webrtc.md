### 编译windows
 
1.生成VS项目文件 

set DEPOT_TOOLS_WIN_TOOLCHAIN=0 

set GYP_GENERATORS=msvs-ninja,ninja 

set GYP_MSVS_VERSION=2015 

python webrtc/build/gyp_webrtc.py

2.编译调试 

以VS2015为例，用VS2015打开all.sln 

将webrtc项目下example下的peerconnection_client设为启动项
